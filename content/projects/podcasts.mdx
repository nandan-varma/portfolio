Podcasts is developed with my friends for hackathon but later updated to be full stack application.

## Features

- Store audio in database
- Play and download audio files
- Player controls for online playing
- login system for favorites

## What I learnt

- React Contexts
- MongoDB
- Communication and Work allotment

## Built with

- [Next.js](https://nextjs.org)
- [chakra-ui](https://chakra-ui.com)
- Deployed on [Vercel](https://vercel.com)
- Database on [mongodb](https://www.mongodb.com)
- Authentication on [Firebase](https://firebase.google.com)
