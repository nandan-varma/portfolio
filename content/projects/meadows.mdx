## Chess Engine that learned to give moves based on parameters like

- Authority over free squares
- Targeting enemy pieces
- Getting point advantage over Enemy 
- Support own pieces
- Finally targeting enemy king.
- and many more...


## Built with

- Tensorflow ( for models )
- Flask ( for API calls )
