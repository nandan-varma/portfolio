# Replog
Replog is a fitness app to track your workouts and exercises.
With privacy and open source in mind, Replog is built to be lightweight and fast.

# Features

**Non-functional requirements:**
- Simple and intuitive UI
- Privacy focused
- Open source
- Cross platform
- Offline support
- Lightweight

**Functional requirements:**
- Display all exercises with important information with images and descriptions
- Track workouts and exercises
- Track progress over time
- Set goals and reminders
- Create custom workout plans
- Calendar view to see past workouts
- AI assistance for workout suggestions
